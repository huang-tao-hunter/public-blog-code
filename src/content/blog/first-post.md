# 欢迎来到我的博客！

这是我的第一篇博客文章。在这里，我会记录一些日常的学习心得、技术分享和生活感悟。

## 关于这个博客

这个博客使用 **Next.js** 构建，部署在 **Vercel** 上。技术栈包括：

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- MDX (支持 Markdown + React 组件)

## 我想写些什么

- 💻 技术文章：编程学习、项目经验
- 📚 读书笔记：最近读的好书
- 🌍 生活感悟：日常中的小思考
- 🎯 目标记录：阶段性目标与复盘

## 开始写作

从今天开始，我会定期更新内容。希望能通过写作，更好地整理思路，也欢迎大家交流讨论！
